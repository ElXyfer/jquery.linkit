# jquery.linkit #
**LinkIt** is a simple jQuery Plugin that allows you to attach a link to any element 

##Version##
0.2.0

##Usage##
    $('span').linkIt({
				    href: 'http://test.com',
				    text: 'Click Me',
				    target: '_blank'
	});

##Venders##
JQuery - [http://jquery.com](http://jquery.com)
